# Brand Styles

每个 HyperFrames 或 Remotion 场景在制作前都必须绑定一个已批准的品牌样式文件。样式文件定义颜色、字体、排版、安全区、运动和禁止模式，避免各镜头自行发明视觉语言。

当前默认样式为 `brand_styles/autocar_default.yaml`。这是 Foundation 规范，不代表字体文件已安装；进入渲染阶段前必须验证字体可用性和授权。

Phase 4 增加 BMW、Mercedes-Benz、Tesla、BYD、Xiaomi Auto 和 Porsche 的编辑视觉参考。这些文件不是官方品牌手册，不授权复制商标，也不得暗示品牌背书。使用前必须通过自动内容适配、素材权利和品牌准确性门禁；权利状态不明确时直接阻断。
