# Factory Shots

用于制造、产能、工艺和供应链报道。不得用无关工厂素材冒充具体企业或生产线。

| 镜头名称 | 用途 | 推荐焦段 | 推荐运镜 | 适用场景 |
|---|---|---:|---|---|
| 生产线环境建立镜头 | 交代厂房规模和工序关系 | 20–28 mm | 缓慢横移或锁定 | 产能、投产、停产、工厂升级 |
| 机器人装配镜头 | 解释自动化动作和制造环节 | 35–50 mm | 平行跟踪或稳定摇摄 | 焊装、涂装、总装技术 |
| 质量检查特写 | 展示检测动作、仪器或人工检查 | 70–100 mm | 锁定或短距离推近 | 质量、安全、召回背景说明 |

执行要求：记录工厂、时间和工序来源；劳动者面部、工牌和屏幕信息必须获得授权或脱敏。
