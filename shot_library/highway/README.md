# Highway Shots

用于高速出行、补能网络、道路政策和长途体验。镜头不得鼓励超速或危险驾驶。

| 镜头名称 | 用途 | 推荐焦段 | 推荐运镜 | 适用场景 |
|---|---|---:|---|---|
| 高速环境建立镜头 | 交代道路、天气和交通密度 | 24–35 mm | 稳定前移或锁定 | 长途续航、高速政策、道路安全 |
| 合法巡航跟车镜头 | 展示车辆与车流的实际关系 | 70–120 mm | 安全距离跟踪 | 能耗、辅助驾驶、舒适性背景 |
| 服务区与补能镜头 | 展示补能设施和停靠场景 | 35–50 mm | 缓慢横移或锁定 | 充电网络、节假日出行、补能效率 |

执行要求：记录道路许可、拍摄方式和安全措施；禁止手持驾驶拍摄、危险跟车和通过剪辑伪造速度。
