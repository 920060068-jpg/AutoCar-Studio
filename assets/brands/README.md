# Brand Assets

保存获准使用的品牌标识、官方媒体包和品牌视觉参考。品牌官方发布不自动等于商业使用授权。

每项文件必须记录权利人、授权用途、平台、地域、有效期、署名要求和证明路径。禁止重绘、裁切水印或修改标识来规避许可，也不得把本目录内容作为新闻事实来源。

Asset Library V1 的标准品牌为：`bmw`、`mercedes`、`audi`、`tesla`、`byd`、`xiaomi`、`toyota`。每个品牌固定使用 `exterior/`、`interior/`、`driving/`、`factory/`、`launch/`、`logo/` 六类目录。

品牌目录名必须使用以上小写规范值；具体车型写入文件名、`database/assets.json.vehicle` 和 `tags`，不得通过新建车型目录破坏统一检索结构。
