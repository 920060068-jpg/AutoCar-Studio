# Daily Production Pipeline

Daily Startup Check -> Topic -> Fact Check -> Script Draft -> Storyboard Draft -> Narration Timing Budget -> Audio-Fit Script -> Number Semantic Classification -> Pronunciation Resolution -> TTS Text Generation -> Voice Speed Calibration -> Static Audio Duration Estimate -> Audio Preflight -> Director Review -> Asset/Editorial Visuals -> Picture Render -> Picture Review -> Picture Lock -> MiniMax Live Probe -> MiniMax TTS -> Measured Audio Fit -> Audio Mix -> Final Render -> Final QC -> Release Candidate.

The daily controller must not call formal MiniMax TTS unless Audio Preflight has passed. Normal narration overrun is handled by automatic condensation, timing budget regeneration and local measured fit. It is blocked only when essential facts cannot fit without changing meaning, MiniMax or the configured voice is unavailable, two automatic revisions fail, audio is corrupt, or verified data conflicts.
