# Original Editorial Visual Mode 规则

## 触发与边界

仅当外部图片和视频均无法合法取得，且已有 blocked 素材报告时启用。启用后停止素材搜索和 PR 等待，不使用官方图片、官方视频、网页截图、搜索缩略图、第三方水印素材或拟真 AI 车型图。

## 允许视觉

- 绑定 verified data 的原创数据动画与时间线。
- 明确标注“示意”或以非实拍风格表达的原创汽车轮廓、空间布局、结构关系和流程图。
- 原创产品关系图、动态图形排版、图标场景和互动选择动画。

任何原创轮廓或机制示意都不得称为官方实车、官方结构图、官方工程图或独立测试结果。不得出现未经核验的尺寸、价格、性能、销量、上市日期或竞品结论。

## 专属视觉门禁

- 混合素材模式的“连续程序化画面不得超过5秒”不适用。
- 同一种视觉构图不得连续超过5秒。
- 每3秒至少发生一次构图、景别、信息层级、动画类型或空间方向的明显变化。
- 连续两个镜头不得复用相同版式。
- 禁止白底卡片式 PPT、文字堆叠、一句话一页和无目的图标阵列。
- 每镜只表达一个核心信息。
- 至少包含1个 Hero 动画、2个数据动画、2个空间或结构示意、1个产品关系图和1个强结尾互动。

## 状态门禁

每镜必须有程序化首帧预览、事实依据、verified data 绑定、原创状态、披露要求和误导风险。自动检查通过后状态为 `ready_for_picture_review`；预览和分镜未自动批准前不得渲染。
